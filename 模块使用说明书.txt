ASSIT-NAT 模块使用说明书
=========================
ESP8266 瑞士军刀·网络工具箱 — 63 模块完整指南


按键通用规则
────────────
  短按 UP     上移光标 / 增大数值
  短按 DOWN   下移光标 / 减小数值
  短按 OK     确认 / 进入 / 执行主操作
  长按 OK     返回上一级 (600ms)
  双击 OK     返回上一级 (部分模块)
  双击 UP     跳到列表头 (部分模块)
  双击 DOWN   跳到列表尾 (部分模块)

  ⚠ 攻击类模块运行时按任意键 = 紧急停止


═══════════════════════════════════════════════════════════════
一、Network Recon (网络侦察) — 16 个模块
═══════════════════════════════════════════════════════════════

1. WiFi Scan (WiFi 扫描)
   ────────────────────
   功能: 扫描周围 WiFi 网络，按信号强度排序显示
   操作: UP/DOWN 浏览列表, OK 重新扫描
   显示: SSID / 信道 / 信号强度(dBm) / 加密类型
         加密: O=无加密 W=WEP A=WPA 2=WPA2 3=WPA3

2. Heatmap (信道热力图)
   ────────────────────
   功能: 显示 1-13 信道上的 AP 分布密度
   操作: UP 切换自动刷新, OK 手动扫描

3. Probe Sniffer (探针嗅探)
   ────────────────────────
   功能: 抓取设备发出的 WiFi Probe Request
   操作: UP/DOWN 浏览捕获的探针

4. Host Discovery (主机发现)
   ─────────────────────────
   功能: ARP 扫描局域网内在线设备
   操作: UP/DOWN 浏览主机列表, OK 开始扫描
   显示: IP 地址 / MAC 地址

5. GW Monitor (网关监控)
   ─────────────────────
   功能: 持续 Ping 网关，监控延迟和丢包
   操作: OK 重置统计数据
   显示: 最小/最大/平均延迟, 丢包率

6. Port Scan (端口扫描)
   ────────────────────
   功能: 扫描目标主机开放端口
   操作: UP/DOWN 浏览端口列表, OK 开始扫描

7. Traceroute (路由追踪)
   ─────────────────────
   功能: 追踪到目标 IP 的路径
   操作: OK 开始追踪
   显示: 每一跳的延迟和 IP

8. Net Stats (网络统计)
   ─────────────────────
   功能: 实时监控网络收发速率
   操作: OK 重置计数器
   显示: RX/TX 字节数, 数据包数

9. DHCP Find (DHCP 发现)
   ─────────────────────
   功能: 发送 DHCP DISCOVER 寻找 DHCP 服务器
   操作: OK 发送请求

10. mDNS Enum (mDNS 枚举)
   ─────────────────────
   功能: 扫描局域网 mDNS/Bonjour 服务
   操作: UP/DOWN 浏览服务, OK 发送查询

11. UPnP Find (UPnP 发现)
   ─────────────────────
   功能: SSDP 发现局域网 UPnP 设备
   操作: UP/DOWN 浏览设备, OK 发送 M-SEARCH

12. DNS Lookup (DNS 查询)
   ─────────────────────
   功能: 手动输入域名进行 DNS 解析
   操作: UP/DOWN 切换字符, 长按 OK 移动编辑光标
         双击 UP 插入字符

13. HTTP Client (HTTP 客户端)
   ──────────────────────────
   功能: HTTP GET 请求，含 4 个预设 URL
   操作: UP/DOWN 切换预设 URL, OK 发起请求
         长按 OK 移动编辑光标

14. Ping Monitor (Ping 监控)
   ─────────────────────────
   功能: 持续 Ping 目标地址
   操作: OK 重置统计数据

15. NTP Query (NTP 查询)
   ────────────────────
   功能: 查询 NTP 时间服务器
   操作: UP/DOWN 调整时区偏移 (±1 小时), OK 查询

16. Wake On LAN (网络唤醒)
   ───────────────────────
   功能: 发送 Wake-on-LAN 魔术包唤醒目标设备
   操作: UP/DOWN 调整 MAC 字节, 长按 OK 移动编辑位, OK 发送

19. RSSI Distance (RSSI 距离估算)
   ──────────────────────────────
   功能: 通过信号强度估算距离
   操作: OK 切换编辑模式 (0:N系数 1:参考RSSI 2:只显示)
         UP/DOWN 调整参数

20. Channel Quality (信道质量)
   ───────────────────────────
   功能: 评估每个 WiFi 信道的质量
   操作: OK 重新扫描

21. RSSI Histogram (信号直方图)
   ────────────────────────────
   功能: 按信号强度区间显示 AP 数量柱状图

22. Spectrum View (频谱视图)
   ─────────────────────────
   功能: 全频谱瀑布图

23. Channel Hopper (信道跳频)
   ──────────────────────────
   功能: 快速切换信道监测信号
   操作: UP/DOWN 调整跳频速度 (5 档), OK 开始/停止

24. Signal Monitor (信号监控)
   ──────────────────────────
   功能: 实时监控指定 AP 的 RSSI 变化
   操作: OK 重新扫描

25. RF Noise (射频噪声)
   ────────────────────
   功能: 测量每个信道的背景噪声
   操作: UP 重置测量, OK 开始/停止


═══════════════════════════════════════════════════════════════
三、WiFi Attack (WiFi 攻击) — 1 个统一模块
═══════════════════════════════════════════════════════════════

   ⚠ 警告: 仅用于授权测试，攻击自己的设备！
   ⚠ 攻击运行时按任意键 = 紧急停止

26. WiFi Attack (WiFi 攻击中心)
   ────────────────────────────
   功能: 统一的 WiFi 攻击平台，集成扫描、选择、攻击全流程

   主菜单 (6 项):
     [1] Scan Targets     → 扫描周围 WiFi，获取目标列表
     [2] Select Target    → 从扫描结果中选择要攻击的 AP
     [3] Attack Mode       → 选择攻击类型 (8 种可选)
     [4] Configure         → 配置攻击参数 (发包数/持续时间等)
     [5] LAUNCH >>         → 二次确认后开始攻击，显示实时动画
     [6] Device Tests      → 设备压力测试

   攻击类型 (Attack Mode):
     Deauth Attack    — 发送解认证帧，踢设备下线 (可调发包数 1-50)
     Beacon Flood     — 广播大量虚假 WiFi 信标 (可调持续时间 1-60s)
     PMKID Capture    — 被动捕获 PMKID 哈希，可保存到文件
     Handshake Capt.  — 被动捕获 WPA/WPA2 握手包
     Rogue AP         — 创建无密码伪基站，追踪连接客户端
     Karma Attack     — 自动响应所有 Probe Request 诱骗设备
     Evil Twin        — 克隆目标 SSID 创建钓鱼热点
     Device Tests     — 设备压力测试 (9 种测试类型)

   操作: UP/DOWN 浏览菜单/调整参数, OK 确认/进入
         攻击中按任意键紧急停止, 超时自动停止
         长按 OK 返回上一级

四、Crypto Tools (加密工具) — 7 个模块
═══════════════════════════════════════════════════════════════

36. TOTP Token (动态口令)
   ──────────────────────
   功能: 生成基于时间的 6 位一次性验证码 (TOTP)
   操作: UP/DOWN 调整位数 (6/7/8)
   显示: 大字体验证码, 30 秒倒计时条, 剩余秒数

37. Password Gen (密码生成器)
   ──────────────────────────
   功能: 生成随机强密码
   操作: UP/DOWN 调整长度 (8-32)
         双击 UP 切换大小写, 双击 DOWN 切换特殊字符, OK 重新生成

38. MAC OUI (MAC 厂商查询)
   ───────────────────────
   功能: 通过 MAC 地址前 3 字节查询设备厂商
   操作: UP/DOWN 调整 MAC 字节, OK 移动编辑位, 长按 OK 复位

39. Base64 (Base64 编解码)
   ──────────────────────
   功能: Base64 编码/解码
   操作: UP/DOWN 切换字符, 长按 OK 移动光标
         双击 UP 插入字符, 双击 DOWN 切换编码/解码模式, OK 执行

40. Hex Convert (十六进制转换)
   ───────────────────────────
   功能: ASCII 与十六进制互转
   操作: UP/DOWN 切换字符, 长按 OK 移动光标
         双击 UP 插入字符, 双击 DOWN 切换模式, OK 转换

41. Hash Calc (哈希计算)
   ─────────────────────
   功能: SHA1 / SHA256 哈希计算
   操作: UP/DOWN 切换字符, 长按 OK 移动光标
         双击 UP 插入字符, 双击 DOWN 切换算法, OK 计算

42. XOR Encrypt (异或加密)
   ──────────────────────
   功能: XOR 对称加密/解密
   操作: UP/DOWN 切换字符, 长按 OK 移动光标
         双击 UP 切换编辑输入/密钥, 双击 DOWN 插入字符, OK 执行

43. Logic Probe (逻辑探针)

48. UART Monitor (串口监视器)
   ─────────────────────────

49. SPI Scan (SPI 扫描器)

   功能: 扫描 SPI 总线设备

50. Freq Generator (频率发生器)

七、433MHz RF (射频收发) — 1 个统一模块
═══════════════════════════════════════════════════════════════

   硬件: SYN480R 接收 (D6) + F115 发射 (D7)

27. 433MHz Tool (433 射频工具箱)
   ─────────────────────────────
   功能: 统一的 433MHz 射频工具，集成扫描/浏览/发射/编辑全流程

   主菜单 (6 项):
     [1] Scan 433MHz      → 扫描捕获 433MHz 信号
                             显示脉宽、协议猜测(PT2262/EV1527等)、位长
                             自动添加到捕获列表, OK 清零计数
     [2] Browse Captures  → 浏览已捕获的码列表
                             UP/DOWN 切换, 显示值/位长/序号
     [3] Send / Replay    → 选择协议后发射当前码
                             UP/DOWN 切换协议, OK 发射
                             屏幕显示发射计数
     [4] Raw Bit Edit     → 位级编辑二进制码
                             3 种编辑模式 (长按 UP/DOWN 切换):
                               Bit    — 逐位编辑, OK 翻转
                               Proto  — 切换协议 1-7
                               Pulse  — 调整脉宽 ±10us
                             OK 发射当前码
     [5] Device Database  → 12 种常见设备码库
                             UP/DOWN 浏览, OK 查看详情
                             详情中 OK 发射
     [6] Temp Sensor      → 被动监听解码温湿度传感器
                             EV1527/DHT 编码, 大字体显示温度
                             显示湿度/传感器类型/数据老化时间

   操作: 主菜单 UP/DOWN 选模式 OK 进入
         各模式内 OK 执行操作, 长按 OK 返回主菜单
         RX/TX 模式自动切换, 无需手动管理

73. Auto Config (自动配置)

附录: 常见操作速查
