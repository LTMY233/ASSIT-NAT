ESP8266 Swiss Army Knife — 接线图
=====================================

一、引脚总览
────────────
  NodeMCU 引脚   GPIO      用途
  ─────────────────────────────────────────
  D0             GPIO16    未使用
  D1             GPIO5     OLED SCL (I2C)
  D2             GPIO4     OLED SDA (I2C)
  D3             GPIO0     按键 UP   ┐
  D4             GPIO2     按键 DOWN ├─ 内部上拉, 按下接 GND
  D5             GPIO14    按键 OK   ┘
  D6             GPIO12    433MHz 接收 (SYN480R DATA)
  D7             GPIO13    433MHz 发射 (F115 DATA)
  D8             GPIO15    PWM 信号输出
  A0             ADC0      逻辑笔 / ADC 电压表 (0~1V)
  3V3            —         供电: OLED, 433MHz 模块
  GND            —         公共地

三、各部件详细接线
──────────────────

3.1 SSD1306 OLED (128×64, I2C, 地址 0x3C)
    OLED 引脚      接到 ESP8266
    ─────────      ────────────
    VCC        →   3V3
    GND        →   GND
    SDA        →   D2 (GPIO4)
    SCL        →   D1 (GPIO5)


3.2 按键 (3个, 轻触开关, 使用内部上拉)
    按键         一脚                   另一脚
    ────         ────                   ────
    BTN_UP   →   D3 (GPIO0)            GND
    BTN_DOWN →   D4 (GPIO2)            GND
    BTN_OK   →   D5 (GPIO14)           GND

    注意: D3/GPIO0 与板载 LED 共用, 按下 LED 亮是正常现象
          D4/GPIO2 也与板载 LED 共用


3.3 433MHz 无线模块

   ★ 如何区分两个模块:

      观察板上的晶振编号 (银白色金属壳元件上的刻字)
      ┌──────────────────────┬──────────────────────┐
      │   晶振标 "13.560"     │   晶振标 "6.7458"     │
      │   板上有 8脚芯片      │   板上有三极管         │
      │   元件多、板子大      │   元件少、板子小       │
      │        ↓             │        ↓             │
      │   SYN480R 接收模块    │   F115 433M 发射模块  │
      └──────────────────────┴──────────────────────┘

    SYN480R (超外差接收)            F115 433M (ASK/OOK 发射)
    晶振: 13.560 MHz               晶振: 6.7458 MHz
    ┌─────────────────┐            ┌─────────────────┐
    │  SYN480R        │            │  F115 433M      │
    │  1 VCC   2 GND  │            │  1 VCC   2 GND  │
    │  3 DATA  4 ANT  │            │  3 DATA  4 ANT  │
    └──┬───┬───┬───┬──┘            └──┬───┬───┬───┬──┘
       │   │   │   │                  │   │   │   │
       │   │   │   │                  │   │   │   └── 焊 17.3cm 导线(天线)
       │   │   │   │                  │   │   └── D7 (GPIO13)
       │   │   │   └── 焊 17.3cm 导线(天线)
       │   │   └── D6 (GPIO12)       │   └── GND
       │   └── GND                   └── 3V3
       └── 3V3

    天线: 433MHz 波长 ≈ 69cm, ¼波长 ≈ 17.3cm
    直接用一根 17cm 的杜邦线焊到 ANT 焊盘即可

    ⚠ SYN480R 规格书写 5V，但 3.3V 供电实测可用
      F115 工作电压范围 2V-12V，3.3V 足够


3.4 PWM / ADC / 逻辑笔
    D8 (GPIO15)  →  PWM 输出 (可接 LED、蜂鸣器、示波器)
    A0           →  逻辑笔探头 / 电压表输入 (0~1V 量程)

    ⚠  ESP8266 ADC 量程仅 0~1V！测量更高电压需电阻分压。


四、面包板快速布局
──────────────────
    面包板顶部电源轨 → 3V3 (ESP8266 3V3 引出)
    面包板底部地轨   → GND

        [OLED]     VCC→3V3轨  GND→地轨  SDA→D2  SCL→D1

     [SYN480R RX]  VCC→3V3轨  GND→地轨  DATA→D6

      [F115 TX]    VCC→3V3轨  GND→地轨  DATA→D7

      [BTN_UP]     一脚→D3    另一脚→地轨
      [BTN_DOWN]   一脚→D4    另一脚→地轨
      [BTN_OK]     一脚→D5    另一脚→地轨


五、供电
────────
    USB 5V → NodeMCU 板载稳压 → 3V3 输出
    ├─ OLED      ~20mA
    ├─ 433 RX    ~5mA
    ├─ 433 TX    ~20mA (发射时)
    └─ ESP8266   ~80mA
    总计约 125mA，USB 500mA 绰绰有余


六、烧录说明
────────────
    用 MicroUSB 线直接连接电脑和 NodeMCU (板载 CH340 USB-TTL)
    PlatformIO 自动处理进入烧录模式，无需手动操作。

    若需手动进入下载模式:
    按住 OK 键 → 按一下 RST → 松开 OK


七、按键操作
────────────
    短按 UP     光标上移 / 上一个
    短按 DOWN   光标下移 / 下一个
    短按 OK     确认 / 进入模块
    长按 OK     返回 (600ms)
    双击 OK     返回 (在模块内)
    双击 UP     跳到列表头
    双击 DOWN   跳到列表尾
